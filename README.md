# Hack Your Device - Interactive Prank Game

## 🎮 How to Play
1. Open `hack.html` in your browser.
2. **Main Button**: Click to \"inject virus\" – progresses through hack levels (3 stages), earns score.
3. **Dodge Button**: Hover to chase – earns small points, moves faster per level.
4. **Goal**: Max score! Tracks highscore in browser storage. Reload to replay.

## ✨ Features Added
- **Multi-level hack sequence** with alerts & progression.
- **Particle effects** (trails, explosions, confetti).
- **Custom virus cursor trail** with pulse glow.
- **Live terminal logs** with timestamps.
- **Score & highscore** persisted with localStorage.
- **Glitch/shake animations**, matrix rain background.
- **Responsive** on mobile/desktop.
- **Sounds** (beeps/glitches – may need user gesture).

## Tech
- Vanilla HTML/CSS/JS – no dependencies.
- Canvas particles, requestAnimationFrame loop.
- Event-driven interactivity.

**Made interesting:** From basic prank to replayable hacker sim game! 🚀

Open with: `hack.html` in browser or `start hack.html` (Windows).
