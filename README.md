# ☠️ Hack Your Device – Interactive Prank Game

## 🧠 Overview
**Hack Your Device** is an interactive prank web game built with **HTML, CSS, and JavaScript**.  
Turn a basic “don’t click” joke into a **multi-level hacker simulation** with scores, animations, and fun effects.  

---

## 🎮 How to Play
1. Open `hack.html` in your browser.  
2. **Main Button** – Click to “inject virus”: progresses through hack levels (3 stages) and earns points.  
3. **Dodge Button** – Hover to chase: earns small points and moves faster with each level.  
4. **Goal** – Max your score! Highscore is tracked in **browser storage**. Reload to replay.  

---

## ✨ Features
- Multi-level hack sequence with alerts & progression.  
- Particle effects: trails, explosions, confetti.  
- Custom virus cursor with glowing pulse.  
- Live terminal-style logs with timestamps.  
- Score & highscore persisted via `localStorage`.  
- Glitch/shake animations and matrix rain background.  
- Fully responsive for mobile & desktop.  
- Sound effects (beeps & glitches).  

---

## 💻 Tech Stack
- **HTML, CSS, JavaScript** (Vanilla, no external dependencies)  
- Canvas particle effects with `requestAnimationFrame`  
- Event-driven interactive gameplay  

---

## 📁 File Structure
- `hack.html` – Main game interface  
- `hack.css` – Styles, animations, gradients, layout  
- `hack.js` – Button interactions, game logic, particle effects  
- `assets/` – Optional folder for sounds, images, and animations

## 📍 How to Run
1. Clone or download the repository.
